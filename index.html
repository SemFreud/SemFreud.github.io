1.每个计算任务都是一个服务，每个服务都有自己的任务路由地址(gateway路由地址)

软件架构: SpringCloud + SpringCloud Gateway + web + Eureka + redis + mysql
责任描述： 负责模块: 定时任务模块、日志模块
